# Rule-Based AI Chatbot 

## Project Overview
This project is a simple Rule-Based AI Chatbot developed using Python.  
The chatbot interacts with users and provides responses based on predefined rules and conditions.

## Objective
The main objective of this project is to understand basic chatbot logic, user input processing, and decision-making using Python.

## Features
- Greets users with welcome messages
- Handles basic user queries
- Provides project-related information
- Supports exit commands like "bye" and "exit"
- Runs continuously until the user ends the conversation

## Technologies Used
- Python

## Concepts Used
- Functions
- While Loop
- If-Else Conditions
- String Handling
- User Input Processing

## Working Process (IPO Model)

**Input:**  
User enters a message through the chatbot.

**Process:**  
The chatbot analyzes the input and matches it with predefined rules using if-else conditions.

**Output:**  
The chatbot provides an appropriate response based on the matched rule.

## How to Run the Project

1. Install Python on your system.
2. Download or clone this repository.
3. Run the following command:

python chatbot.py

## Sample Interaction

User: Hello  
Bot: Hello! How can I assist you today?

User: Tell me about project  
Bot: Project 1 is a Rule-Based Chatbot using IPO model.

User: Bye  
Bot: Goodbye! Have a great day.

## Conclusion
This project demonstrates the basics of building a rule-based chatbot using Python and helps in understanding automation, logic building, and conversational systems.
